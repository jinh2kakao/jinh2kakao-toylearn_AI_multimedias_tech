## 프롬프트 유형과 선택 이유

### 1. [핵심 기반] 3. 역할 부여 (Role-Playing)
* **선택 이유:** 이 문제는 일반적인 조언이 아닌, 유튜브 범퍼 광고와 퍼포먼스 마케팅에 대한 깊은 이해가 필요합니다. AI에게 **"데이터 기반 유튜브 광고 크리에이티브 전략가"** 또는 **"퍼포먼스 마케팅 전문 PM"** 역할을 부여하여, 숫자에 숨겨진 의미를 해석하고 전문적인 해결책을 도출하도록 합니다.
* **효과:** "할인 광고가 인기가 없네요"가 아닌, "신규 브랜드 런칭 초기, 인지(A, B)가 쌓이기 전에 구매(C)를 강요하는 메시지는 3초 이내 이탈을 유발합니다"와 같은 전문가 수준의 진단을 얻습니다.

### 2. [핵심 엔진] 6. 체인 오브 스루트 (Chain of Thought, CoT)
* **선택 이유:** 훈련 초점인 **'원인 분석'**을 수행하는 데 가장 강력한 도구입니다. AI가 "C 영상의 3초 이탈률이 80%"라는 데이터를 보고, "왜?"라는 질문을 스스로 단계별로 하도록 유도합니다.
* **효과:** AI가 '데이터(3초 이탈) → 현상(즉각적 거부감) → 가설(1. 과도한 세일즈, 2. 초반 후킹 실패, 3. A/B 대비 이질감) → 결론(신규 브랜드에겐 할인보다 가치 소구가 먼저)'이라는 논리적 추론 과정을 거쳐 근본 원인을 도출하게 만듭니다.

### 3. [행동 지시] 1. 지시형 (Instructional)
* **선택 이유:** 원인 분석(Why)이 끝났다면, 구체적인 해결책(What)을 명령해야 합니다. "원인을 분석해 줘"에서 더 나아가, "즉시 실행할 수 있는 C 영상 **재편집(re-edit) 기획안** 2가지를 제안해 줘"라고 명확한 결과물을 지시합니다.
* **효과:** AI가 분석에서 멈추지 않고, 내일 회의에서 바로 제안할 수 있는 실질적인 대안(예: A 영상의 후킹 + C 영상의 할인, B 영상의 후기 + C 영상의 할인 조합)을 생성합니다.

### 4. [출력 규격] 4. 제약 조건 (Constraint-Based)
* **선택 이유:** AI의 답변은 클라이언트에게 보고할 '긴급 대응 보고서' 형태여야 합니다. "내일 오전 회의용", "다음 목차에 맞춰 작성" 등 명확한 형식과 제약(예: 추가 촬영 불가, 6초 유지)을 부여하여, 즉시 활용 가능한 비즈니스 문서를 생성합니다.
* **효과:** 산만한 아이디어 나열이 아닌, 구조화된 보고서를 통해 PM(사용자)의 작업 시간을 획기적으로 단축시킵니다.


## 작성 위해 추가 필요 요건


    **C 영상(6초)의 구체적인 스토리보드 (특히 첫 3초):** 3초 이탈의 원인을 알려면 첫 3초에 무엇이 나왔는지 알아야 합니다.
    * *해결책:* 프롬프트에서 AI에게 "3초 이탈률이 80%인 범퍼 광고가 첫 3초에 저지를 수 있는 **가장 치명적인 크리에이티브 실수 3가지**를 가설로 설정하라"고 지시합니다. (예: 1. 'BIG SALE' 텍스트가 0초부터 등장, 2. 브랜드 로고가 너무 김, 3. A/B와 톤앤매너가 다른 모델 사용)


## 가상 문제 상황
1. 가상 문제 상황
페르소나: 광고 대행사의 콘텐츠 제작 팀장
배경: 당신은 클라이언트 'A'(신규 뷰티 브랜드)의 신제품 런칭을 위해 3가지 버전의 유튜브 범퍼 광고(6초) 영상을 제작하여 집행했습니다. 3가지 영상은 각각 다른 소구점(A: 성분 강조, B: 사용 후기, C: 할인 혜택)을 강조했습니다.
핵심 문제: 캠페인 2주차 데이터 분석 결과, A와 B 영상은 평균 클릭률(CTR) 2.5%, **구매 전환율 1.8%**를 기록하며 순항 중입니다. 하지만 **가장 많은 예산을 투입한 C 영상(할인 혜택 강조)의 CTR은 0.8%**에 불과하며, **구매 전환율은 0.1%**로 처참한 수준입니다. 특히 C 영상은 3초 이내 이탈률이 80%에 육박합니다.
해결 과제: 클라이언트는 C 영상의 즉각적인 중단과 예산 환불까지 고려하고 있습니다. 당신은 C 영상의 성과가 저조한 구체적인 원인을 분석하고, 이 영상을 즉시 수정(re-edit)하거나 대체할 수 있는 긴급 대응 방안을 내일 오전 회의까지 보고해야 합니다.
2. 훈련 초점 (힌트)
이 문제는 [원인 분석] 능력을 평가

## 프롬프트 실행 결과

**[긴급 대응 보고서: C 영상(할인 혜택) 성과 개선안]**

### 1. 데이터 요약 및 문제 정의

A, B 영상(가치 소구)은 CTR 2.5%, 전환 1.8%로 성공적이나, C 영상(혜택 소구)은 CTR 0.8%, 전환 0.1%로 부진합니다. 특히 C 영상의 **'3초 이탈률 80%'**는 이 영상이 시청자에게 첫 3초 안에 즉각적인 거부감을 주고 있음을 시사합니다.

### 2. C 영상 실패 원인 심층 분석 (Root Cause Analysis)

A/B의 성공은 시청자가 '신규 뷰티 브랜드'에 대해 "이게 뭐지?"(성분, 후기)라는 궁금증을 가지고 있음을 보여줍니다. 반면 C 영상의 실패 원인 가설은 다음과 같습니다.

* **가설 1: 맥락 없는 할인 제시 (Contextless Offer)**
    신규 브랜드의 제품 가치(A, B)를 인지하기도 전에, 첫 3초부터 '할인'이라는 가격 정보를 제시하여 시청자가 이를 '스팸 광고'로 인식하고 즉시 스킵(이탈)했습니다.
* **가설 2: 후킹 실패 및 정보 과부하**
    6초라는 짧은 시간에 '제품'과 '할인 혜택'을 모두 보여주려다 보니, 첫 3초에 제품의 매력이나 브랜드 톤앤매너가 아닌 'SALE%' 텍스트가 화면을 장악하여 시각적 피로감과 이탈을 유발했을 가능성이 큽니다.
* **가설 3: 타겟팅 불일치**
    A, B 영상이 '성분/후기 탐색' 타겟에게 도달한 반면, C 영상은 '할인 민감' 타겟에게만 집중되었으나, 해당 타겟은 아직 이 브랜드 자체를 몰라 혜택에 반응하지 않았습니다.

### 3. 즉각적인 긴급 대응 방안 (Immediate Action Plan)

#### A. C 영상 긴급 재편집(Re-edit) 기획안 (2가지)

* **기획안 1 (성공 공식 이식: A+C)**
    * **컨셉:** "가치 제시 후 혜택 제시"
    * **기대 효과:** A 영상에서 검증된 '성분' 후킹으로 3초 이탈을 방지하고, 이미 흥미를 느낀 시청자에게 '할인' 혜택을 추가로 제시하여 전환율 극대화.
    * **6초 스토리보드:**
        ```
        A 영상의 핵심 성분 시각화 장면 (후킹)
        제품 클로즈업
        "런칭 기념 20% OFF" (혜택 제시)
        ```

* **기획안 2 (C 소스 순서 변경)**
    * **컨셉:** "제품 선-인지, 혜택 후-제시"
    * **기대 효과:** C 영상의 모델/제품 컷을 먼저 제시하여 제품을 인지시키고, 이탈률이 낮은 4초 이후에 할인 혜택을 노출.
    * **6초 스토리보드:**
        ```
        C 영상의 매력적인 모델 컷 또는 제품 텍스처 컷 (사운드/카피 없음)
        제품명 및 브랜드 로고
        "지금, 20% 할인"
        ```

#### B. A/B 테스트 전략

* 기존 C 영상은 즉시 중단(Pause)합니다.
* 재편집된 C-1(A+C), C-2(C 재구성) 안을 A/B 영상과 동일한 예산으로 1주일간 테스트하여 최고 효율의 크리에이티브를 선별합니다.

### 4. 클라이언트 설득을 위한 핵심 'Talking Point'

* "데이터는 '할인' 메시지가 나쁘다는 것이 아니라, '말하는 순서'가 잘못되었다는 것을 명확히 보여줍니다."
* "시청자는 3초 안에 이탈했습니다. 이는 C 영상의 후반부가 아닌 '첫인상'의 문제입니다. 따라서 첫인상을 A/B처럼 매력적으로 바꾸는 재편집이 필요합니다."
* "C 영상의 예산을 포기하는 것이 아니라, A/B의 성공 데이터를 C에 적용하여 '가장 효율적인 할인 광고'로 재탄생시키겠습니다."
